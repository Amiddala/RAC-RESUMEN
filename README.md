# Redes Avanzadas – Apuntes Interactivos

Este repositorio contiene mis apuntes de la materia **Tecnologia de Redes Avanzadas** de la Universidad Mayor de San Simón (UMSS), organizados y presentados en formato web (HTML) para facilitar su visualización, estudio y despliegue.

---

## Objetivo del proyecto

El objetivo de este proyecto es:

- Transformar apuntes tradicionales (PDF/diapositivas) en una **experiencia web interactiva**
- Facilitar el aprendizaje mediante una presentación más clara y estructurada
- Crear un recurso accesible desde cualquier dispositivo (deploy online)
- Practicar habilidades de desarrollo frontend y organización de contenido técnico

---

## ¿Qué contiene?

Actualmente, el proyecto incluye contenido sobre:

### Redes de Conmutación

- Introducción a la conmutación
- Tipos de conmutación:
  - Conmutación de circuitos
  - Conmutación de paquetes
- Ventajas y desventajas de cada modelo
- Modos de transmisión:
  - Circuitos virtuales
  - Datagramas

---

### Tecnologías de Red

#### ATM (Asynchronous Transfer Mode)

- Arquitectura de red troncal
- Estructura de celdas (53 bytes)
- Capas del modelo ATM
- Canales virtuales (VCI/VPI)
- QoS (Quality of Service)

#### Multiplexación

- TDM (Time Division)
- FDM (Frequency Division)
- CDM (Code Division)
- Aplicaciones en telecomunicaciones e informática

---

### Protocolos y Redes WAN

#### X.25

- Arquitectura y funcionamiento
- Dispositivos (DTE, DCE, PSE)
- Circuitos virtuales (PVC y SVC)
- Protocolos: PLP, LAPB

#### Frame Relay

- Transmisión de tramas
- Funcionamiento basado en circuitos virtuales
- Ventajas frente a X.25

---

### Tecnologías de Acceso

#### ADSL

- Comunicación asimétrica
- Componentes:
  - Modem (ATU-R / ATU-C)
  - DSLAM
  - Splitter
- Velocidades y estándares
